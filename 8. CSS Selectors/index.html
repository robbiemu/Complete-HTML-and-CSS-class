CSS Selectors

Basic means of selection

CSS Style rules are applied to lements that match seleciton rules. These can be literal matches:

p { // all paragraph elements everywhere on the page
    text-size: 1.2em;
}

And multiple seleciton criteria can be supplied to a single style declaration:

p, span { font-family: mono; }

Common attributes used for selection are id and class attributes:

id

#namedElement { ... }

class

.itemsToCenter { // any element with this class.
 ...
}
p.center, img.center { // but not, for example, table.center
 ...
}

Element States

Elements have at least the hover state, which apply to them only when the condition is met. In the case of hover, it is true of the element only while the cursor of the user is above the element. Styling an element which is in a state is specified in the selector like so:

p:hover { ... }
hover over the above code to see it in action.

This brings up the point that CSS declarations are applied in deference to their specificity. For example, give rules for a {}, a.ofClass{}, a#item, a:hover {}, all will apply, and in this case they will be applied from left to right so that the most specific one will apply last.

One of the tags who's states are commonly styled with CSS is the a anchor tag, which takes the following states:

  • a:link - a normal, unvisited link
  • a:visited - a link the user has visited
  • a:hover - a link when the user mouses over it
  • a:active - a link the moment it is clicked
