CSS Box Model

Borders

The border property defines a line wrapping around an element.

An element with width:200px and border: 1px will be 201px wide: the border counts towards the interior space of the element it is applied to (unlike margin).

Note that in the following example the partially transparent border is red (belonging to the child element, instead of grey:

border: rgba(0,255,0,0.3) solid 10px;

Borders can be defined with different colors or different line forms (solid, dashed, dotted, etc). The most likely choice of line form is solid. These can be specified separately, as border-width, border-style, and border-color. The color will inherit the text color of the element by default. A border-radius can also be specified, to round the edges of the element (border).

Each side of a border can be specified separately as border-left, border-top, etc. And the previous can be combined, as in border-left-color.

background property

Elements may have a background property, even if they are inline elements. The background can be specified as an image or as a color.

background: green
background-color: red;
background: url(simcity\ large.png);

there is a lot of complexity in how the image can be positioned and scaled behind the image. It would be good to look up the details.

Float and Clear

The css property float can lift an element from its normal vertical positioning in the text, allowing text to wrap around the object. Normally, inline text flow would break at the object, leading ot a serial display:

This is some text. Following is an element in the flow of text:

and after it, some more text

The float attribute can be used to offset the image to the right or left of the flow of text:

This is some text. Following is an element in the flow of text:

float:left
and after it, some more text

To cause the text to break at the object being floated, and return to a normal flow, it must be given the clearproperty:


this is text before either

float: left

this is text between the two

float: right

this is text after the two

This text has clear:left. It has not cleared the float: right, as you can probably tell.

This text has clear:right

To clear both at once you can use clear:both
