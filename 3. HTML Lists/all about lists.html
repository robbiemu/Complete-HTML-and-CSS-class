Lists

List items are added, regardless of list type, with the li tag.

Ordered List

  1. A first list item
  2. Another list item

Ordered lists support attributes. The type attribute describes the numbering:

  • type="1" is a numbered list, and is the default.
  • type="3" would stat the list at 3, instead of 1.
  • type="a" is an "a,b,c" list.
  • type="i" is a lower case roman numeral list "i,ii,iii" list.
  • etc

The li tag takes the value attribute, which can be used to set the numbering in an ordered list starting at that item with that value. So value="c" in the second item in an alphabetical ordered list would produce a list whose numbering would go: "a, c, d" etc.

note: only ordered lists support the type attribute.
