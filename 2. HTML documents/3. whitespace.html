whitespace

Text with extra spaces in it will only render 1 space. You can see that in the code for this page right here.

The pre tag can be 

		used
		to insert extra    white    space

if it is desired. It adds the text literally, without formatting it, and it is a block element — it automatically will terminate a paragraph element.

If you look at the source for this html page, you will see that text in p tags should render in a serif font. This did not get applied at the pre tag. So another way to add white space, is to use the non-breaking space &nbsp; in your text, or to use the br tag.
