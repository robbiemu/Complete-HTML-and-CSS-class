The image tag

Images can be inserted with the img tag. Here is an example image:

Images are in-line-block elements by default. This means that they will try to merge with the flow of text if they are in a block containing one (such as in a p tag, but will render as block elements (whose spacing is not determined by the flow of text) in any other context. An example of that is the image inline at the start of this sentence!

Attributes

It takes a required src attribute which is a link to an image file, either stored locally along with the page or available online. Any URL will do.

The title attribute is commonly used with the image tag.

The alt attribute will render even when the image does not.

Image tags take width and height attributes. These are pixel values.
