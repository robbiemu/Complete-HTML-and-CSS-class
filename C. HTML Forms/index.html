HTML Forms

Form basics

The form tag creates a form that will post data back to the server, injecting it the last part of the URL. Like tables, the outer tag itself does not render: its contents does.

Form validation requires javascript. Submitting a form will will by default use the URL of the current page. A different base URL can be specified in the form tag with the action attribute.

Input tag

The input tag is used to creat a field for the user to enter data. Its type attribute can be "text" or something else; for example, "password" will take text input and show dots instead of characters.

The value of "file" will allow the user to select a file using their local filesystem, which will only be submitted manually (with javascript) along with the rest of the form data. Other important values are "radio" (for a "radio" binary option), and "checkbox". These are related together by specifying the same name. Radio buttons are meant to be required, and cannot be unchecked (without using javascript). Two more are "button" (for a button to attach to javascript, etc), and "submit" (a special button to submit the form).

The "file" and "password" options will be in the url in plain text (and therefore deeply insecure for passwords) unless you set the method to "post". To support encryption, you have to set annother attribute, like so: enctype="multipart/form-data". Getting HTTPS set up on your webserver is also important, if it is handling secure data. But that is beyond the scope of this tutorial.

In every case, the name of an input specifies the name used to submit the data captured by the input once the form is sumbitted. The value attribute specifies the value submitted (if it is not a text field input, etc).

By default, buttons are styled according to the browser, which usually uses the operating system's design language. However, they can be styled like any other element with CSS.

Textarea

If you want a large text area for input, you can specify one with the textarea tag.

Select and Option tags

Another major tag is the select tag. It encapsulates a group of option tags that together define a dropdown box with its options.

The select tag takes the name attribute, while the option tags take the value attributes, one of which will be selected by the user and submited in the form submission. The default option can be set with an attribute like so: selected="selected"
