Projects on the filesystem

A good idea is to separate the content, css, and javascript into different folders. This creates a filesystem heirarchy like:

  • / - contains index.html, project files, git stub, etc
  • /bin - folder for additional tools used in maintaining the site, like automatic documenation generators, etc
  • /doc - folder for site, code, etc documentation
  • /conf - folder for configuration files
  • /css - folder for CSS throughout the site
  • /js - folder for site javascript
  • /js/unitTests - folder for javascript unitTests (to ensure that all parts of the site are working)
  • /assets - folder for media
  • /assets/images - for example, one specifically for image files.

If you are utilizing a specific web framework, this will be specified for you. And you likely will not often need all of those folders, but creating an empty site like this and then copying it for each new project is a great way to create a standard work environment to start from.

Laying out a page

A good pattern to follow is to encapsulate every area on import in your page. Like in the following:


<html>
  <head></head>
  <body>
    <div id="outerContainer"><!-- renderable part of the body -->
      <div id="header"><!-- top section --></div>
      <div id="nav"><!-- main navigation element --></div>
      <div id="contents"><!-- main body of current page --></div>
      <div id="footer"><!-- legalese and contact info, etc --></div>
    </div>
  </body>
</html>

This encapsulation makes each logical unit easily addressable by CSS or javascript.

In general, web pages have a top bar, a nav element, and a content pane. Many also include a footer, while others simply link to those details from the site navigation.

In order to ensure side panels (like a side nav) have a specific width, or that top bars have a specific height, use CSS. It is important to specify width for all the container divs that divide up th page:


div#outerContainer {
//    padding: 0 1px;
}
div#header {
    height: 1.4em;
}
div#nav, div#contents {
    float: left;
}
div#nav {
    width: 20em;
}
div#contents {
    width: 100%;
}
div#footer {
    clear: both;
}
div#outerContainer, div#header, div#footer {
    width: 100vw;
}
